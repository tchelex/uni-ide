==================================================
  UNI IDE — как установить и запустить (для ученика)
==================================================

1. Запусти установщик  UNI-IDE-Setup-x.x.x.exe  и пройди установку
   (права администратора НЕ нужны — программа ставится в твою папку
   пользователя). На последнем шаге будет две галочки:
      [ ] Установить драйвер CH340  — поставь её, если плату ESP32
          раньше на этом компьютере не подключали;
      [v] Запустить UNI IDE.
   Драйвер нужен, чтобы компьютер «увидел» плату. Он попросит права
   администратора — это нормально, соглашайся.

2. Подключи плату ESP32 к USB.

3. Запусти UNI IDE (ярлык на рабочем столе или в меню «Пуск»).
   Откроется окно среды программирования.

4. В правом верхнем углу нажми кнопку «обновить порты» (значок ↻)
   и выбери COM-порт платы.

5. Пиши код. Кнопки:
   - «Проверить»  — только компиляция            (Ctrl+R)
   - «Загрузить»  — прошить программу на плату    (Ctrl+Enter)
   Смотреть, что плата шлёт в Serial, — вкладка «Serial»
   справа, кнопка «Подключить».

   Проект сохраняется сам. Меню «Файл»:
   - Новый проект        (Ctrl+N)
   - Открыть проект…     (Ctrl+O)
   - Примеры             — готовые примеры из библиотек
                           (открываются копией — портить не страшно)
   - Сохранить как…      (Ctrl+S)
   - Последние проекты   — список недавних.

   Поиск по коду — Ctrl+F, поиск с заменой — Ctrl+H.

--------------------------------------------------
ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
--------------------------------------------------
- Порт не появляется       -> не поставлен драйвер CH340 (шаг 1)
                              или плохой/зарядный кабель без данных.
- При прошивке «Connecting…» -> удерживай кнопку BOOT на плате,
                              когда началась прошивка.
- Первая компиляция долгая  -> это нормально, дальше будет быстрее.

Интернет НЕ нужен — всё работает офлайн.
