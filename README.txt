ДРАЙВЕР USB-UART ДЛЯ ПЛАТЫ ESP32

На платах ESP32 Dev Module для связи по USB стоит чип CH340/CH341.
Его драйвер (CH341SER.EXE) уже лежит в этой папке и автоматически
попадает в установщик: при установке UNI IDE на последнем шаге будет
галочка «Установить драйвер CH340».

Файл драйвера:
  CH341SER.EXE — официальный установщик WCH (CH340/CH341).
  Источник: https://www.wch-ic.com/downloads/CH341SER_EXE.html

Если на вашей плате другой чип (CP2102 / CP210x, Silicon Labs):
  скачайте «CP210x Universal Windows Driver» и поставьте отдельно —
  https://www.silabs.com/developer-tools/usb-to-uart-bridge-vcp-drivers
  (UNI IDE определяет порт CH340 автоматически; CP210x-порт тоже
  появится в списке после установки его драйвера.)

Драйвер ставится с правами администратора и только один раз на каждом ПК.
